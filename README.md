A Power BI dashboard that analyzes mobile sales data with interactive KPIs, maps, charts, and monthly filters to help understand sales performance, customer behavior, and business trends.

Features
📊 Total Sales, Quantity, Transactions & Average Sales KPIs
🗺️ City-wise Sales Analysis using Map Visual
📈 Daily Sales & Quantity Trend Analysis
⭐ Customer Ratings Distribution
💳 Payment Method Analysis (UPI, Cash, Debit Card, Credit Card)
📱 Mobile Model-wise Sales Comparison
🏷️ Brand-wise Sales Summary
📅 Month-wise Interactive Slicer for filtering data
📌 Clean and interactive Power BI dashboard design

⭐Technologies Used
  -> Microsoft Power BI
  -> Power Query
  -> DAX (Data Analysis Expressions)
  -> Data Visualization
